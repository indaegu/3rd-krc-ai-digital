# prototype/ — 인터랙티브 시각 참고물

`mulsigye-app-prototype-v2.html`은 시나리오 네 가지와 핵심 상호작용을 빠르게 확인하는
단일 HTML 참고물입니다. 브라우저에서 열고 왼쪽 컨트롤로 시나리오를 전환할 수 있습니다.

- 제품 행동·수치 의미는 `docs/product.md`, `docs/data-sources.md`, `docs/prediction-model.md`를 따릅니다.
- 토큰·컴포넌트 규칙은 `docs/design-system.md`를 따릅니다.
- 프로토타입이 문서와 다르면 문서를 기준으로 프로토타입을 같은 변경에서 갱신합니다.
- 프로토타입 코드를 제품 코드로 복사하지 않습니다. Next.js와 Compose 구현의 시각 검증에만 씁니다.
