# design-system.md — 디자인 시스템

> UI 작업 전에 읽는다. **시각 스펙의 원본은 `prototype/mulsigye-app-prototype-v2.html`** 이다.
> 이 문서와 프로토타입이 다르면 프로토타입이 이기고, 발견 즉시 이 문서를 고친다.

## 방향

Toss·카카오뱅크 계열: 흰 배경, 단색 카드(그라데이션 금지), 큰 볼드 타이포,
여백 넉넉히, ~해요체 카피. 1차 타깃(고령 농업인) 기준 가독성 우선.

## 토큰 (프로토타입 `:root` 그대로 이식)

```css
--ink:#191F28; --ink2:#4E5968; --ink3:#8B95A1; --ink4:#B0B8C1;
--bg:#FFFFFF; --gray50:#F9FAFB; --gray100:#F2F4F6; --gray200:#E5E8EB;
--blue:#3182F6; --blue-deep:#1B64DA; --blue-tint:#E8F3FF; --blue-soft:#D6E8FF;
/* 가뭄 단계 (fg/bg 쌍) */
--ok-fg:#159570;   --ok-bg:#E6F6F0;    /* 정상 */
--watch-fg:#9A6700;--watch-bg:#FFF3D6; /* 관심 */
--care-fg:#D9510C; --care-bg:#FFEBDE;  /* 주의 */
--alert-fg:#E5372F;--alert-bg:#FDEBEA; /* 경계 */
--crit-fg:#A11C1C; --crit-bg:#F8E2E2;  /* 심각 */
--r-lg:24px; --r-md:18px; --r-sm:12px;
```

- 폰트: Pretendard Variable(셀프호스트) → 폴백 시스템 고딕. 명조 사용 금지.
- 단계 색은 위 5쌍 외 추가 금지. 단계 임계값과 색 매핑은 코드 한 곳에서만.

## 레이아웃·컴포넌트 규칙

- 메인 모듈 간 간격 **24px** 고정. 카드 배경 `--gray50`, radius `--r-lg`, 내부 패딩 20px.
- CTA 버튼: 높이 56px, radius 16px, `--blue`, 굵기 700. 비활성 = `--gray200`.
- 수위 게이지: 눈금(관심/주의/경계/심각)은 **비이커 바깥 오른쪽**에 배치 —
  물이 차도 글자가 가려지면 안 된다. 물 색은 밝은 톤(#98CBFF→#7EB6FA 미세 그라데이션 허용).
- 단계 뱃지: 단색 tint 칩(fg/bg 쌍). 밑줄 강조 금지, 굵기·색으로만 강조.
- 바텀시트: 상단 radius 24px + 그랩바, 딤 rgba(25,31,40,.45).

## 로딩 패턴 4종 (혼용 금지 — 상황별 지정 패턴만)

| 상황 | 패턴 |
|---|---|
| 메인 진입·새로고침 | 모듈별 **스켈레톤**(shimmer 1.3s) — 풀스크린 스피너 금지 |
| 주소 검색 결과 대기 | 리스트 영역 인라인 스피너 |
| 제출성 버튼(등록하기 등) | 버튼 내부 흰 스피너 + busy 잠금 |
| 헤더 타임스탬프 | "불러오는 중…" 텍스트 |

## 애니메이션

- 물 출렁임: 회전 타원 2겹(7s / 11s reverse), 로딩 후 수위 0→목표 **1.6s 차오름**.
- 로고 새로고침: 물방울 **rainfall**(위→아래 반복 낙하 0.62s). 회전 금지.
- 화면 전환: fade + 7% 슬라이드, 0.3s.
- `prefers-reduced-motion`: 출렁임·rainfall·shimmer·전환을 모두 끈다. 신규 애니메이션도
  반드시 이 미디어쿼리에 등록한다.

## 접근성 기준

본문 15px 이상, 핵심 숫자 초대형(3rem급), 대비 AA 이상, 모든 인터랙션 요소에
`:focus-visible` 아웃라인, 아이콘 버튼에 `aria-label`.
