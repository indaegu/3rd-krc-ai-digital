# 물시계 (Mulsigye)

> 농업용수 부족 시점을 예측하고, 농업인에게 지금 해야 할 행동을 알려주는 **AI 물관리 코치**

제3회 KRC AI 디지털 혁신 공모전 · 서비스개발 분야(안전강화) 출품작입니다.

## 무엇이 다른가

기존 포털(농어촌알리미, 국가가뭄정보포털 등)은 **"지금 저수율이 몇 %인지"** 를 보여줍니다.
물시계는 한국농어촌공사 공공데이터와 추세 예측으로 **"이 속도라면 며칠 뒤 다음 가뭄 단계에
도달하는지"** 를 계산하고, **"그 전에 무엇을 해야 하는지"** 를 쉬운 말로 알려줍니다.

- 📊 활용 데이터: 농촌용수 저수지 수위정보 API · 전국 저수지 일별 저수율 · 논가뭄지도 · 가뭄예경보 (공공데이터포털)
- 🤖 AI: 추세 기반 저수율 예측(백테스트로 모델 선정) + LLM 행동요령 생성
- 📱 형태: 반응형 웹 (WebView 앱 셸 대응)
